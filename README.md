# Tauri Yeelight

Desktop app for controlling Yeelight smart bulbs through LAN.

## Protocol

The miIO protocol documentation can be found [here](https://github.com/OpenMiHome/mihome-binary-protocol/blob/master/doc/PROTOCOL.md).

## Development

```bash
npm install
npm run tauri dev
```

## Build

```bash
npm run tauri build
```

## Tech Stack

- Tauri 2
- React
- TypeScript
- Vite
